Superstatic w/ gulpjs

Superstatic is an enhanced static web server. It has fantastic support for HTML5 pushState applications as well as clean URLs and other goodies.

How to use

Just run 'npm start'.

$ npm install && npm start

Options

Sample configuraton ~ superstatic.json

{
  "name": "app-name",
  "root": "./app",
  "routes": {
    "/": "start.html",
    "/user": "userlist.html"
  },
  "clean_urls": false,
  "error_page": "error.html",
  "cache_control": {
    "/*": false,
    "**/*.html": 600
  }
}

More info. See Superstatic Configuration and BrowserSync options.
